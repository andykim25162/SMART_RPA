# 🔑 구글 API 발급 가이드

SmartRPA 시스템에서 구글 시트 연동이나 데이터 분석을 위해 Google Cloud API 키가 필요합니다. 아래 단계를 차례대로 따라해 주세요.

---

### 1단계: Google Cloud Console 접속
먼저 [Google Cloud Console](https://console.cloud.google.com/)에 접속하여 로그인을 완료합니다.

> 💡 **Tip:** 가급적 시스템 전용으로 사용할 구글 계정으로 로그인하는 것을 권장합니다.

### 2단계: 프로젝트 생성
1. 상단 바의 **[프로젝트 선택]**을 클릭합니다.
2. 팝업창 우측 상단의 **[새 프로젝트]**를 누릅니다.
3. 프로젝트 이름(예: `SmartRPA-Project`)을 입력하고 **[만들기]**를 클릭합니다.

### 3단계: API 및 서비스 활성화
1. 왼쪽 메뉴에서 **[API 및 서비스] > [라이브러리]**로 이동합니다.
2. 검색창에 사용하려는 API를 검색합니다. (예: `Google Sheets API`, `Google Drive API`)
3. 해당 API를 선택한 후 **[사용]** 버튼을 클릭합니다.



### 4단계: 사용자 인증 정보 생성 (API 키)
1. 왼쪽 메뉴의 **[사용자 인증 정보]** 탭을 클릭합니다.
2. 상단의 **[+ 사용자 인증 정보 만들기]** -> **[API 키]**를 선택합니다.
3. 생성된 API 키를 복사하여 안전한 곳에 저장해 두세요.

> ⚠️ **Warning:** API 키는 절대 타인에게 노출되지 않도록 주의하세요. 깃허브 코드에 직접 올리지 않는 것이 원칙입니다.

### 5단계: 서비스 계정 생성 (권장)
구글 시트에 직접 데이터를 쓰고 읽으려면 '서비스 계정'이 필요합니다.
1. **[사용자 인증 정보 만들기]** -> **[서비스 계정]**을 선택합니다.
2. 계정 이름을 입력하고 생성한 뒤, 생성된 이메일 주소(예: `...@gserviceaccount.com`)를 복사합니다.
3. **[키]** 탭에서 **[새 키 만들기] (JSON)**를 눌러 파일을 다운로드합니다. 이 파일이 SmartRPA 설정에 사용됩니다.

---

### 💡 마지막 확인 사항
* 발급받은 **JSON 키 파일**을 SmartRPA 프로그램의 `config` 폴더에 넣었나요?
* 구글 시트 오른쪽 상단의 **[공유]** 버튼을 눌러, 위에서 복사한 **서비스 계정 이메일**을 편집자로 추가했나요?
